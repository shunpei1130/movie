# 推し活パスポート MVP

Next.js 14 で作成した、推し活パスポートのMVPデモです。

## 実装済み機能（MVP範囲）

### ファン向け
- ファン登録（LINEログイン / メール+認証コードの簡易選択）
- パスポートID発行
- 応援行動記録
  - 来場（QRトークン入力）
  - 購入（購入コード入力）
  - 視聴（視聴キーワード入力）
- 応援年表表示
  - 初応援日
  - 累計応援回数
  - 参加イベント数 / 購入件数 / 視聴回数
  - 月別履歴（時系列降順）
- 応援タイプ分類
  - 現場型 / 収集型 / 視聴継続型 / バランス型
- シェアカード表示とSNS共有リンク

### 運営向け
- サマリー表示
  - 総登録ファン数
  - 月間アクティブファン数
  - 新規登録数
  - 再接触率
  - 直近30日応援行動数
- ファン一覧
  - 表示名 / 最終応援日 / 累計応援回数 / 応援タイプ / 離脱予兆
  - 検索 / タイプ絞り込み / 離脱予兆フィルタ
- 行動分析
  - イベント別参加者数
  - 商品別記録件数
  - 初回→2回目接触率
  - イベント参加後の再購入率
  - 購入後の再来場率
- 記録設定
  - イベント作成（QRトークン発行）
  - 商品作成（購入コード発行）
  - 視聴設定（視聴キーワード登録）

## データ保持

- 本MVPは `localStorage` に保存します (キー: `oshi-passport-mvp-v1`)。
- サーバーサイドDB連携はMVPの次段階で拡張可能です。
